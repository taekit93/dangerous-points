# 미결 질문 — GeoJSON 장애물 임포트

- [완료] 입력 방식은 파일 업로드만? 텍스트 직접 입력도? -> 파일 업로드(.geojson/.json) + 텍스트 직접 입력 둘 다 지원
- [완료] FeatureCollection에 여러 Feature가 있을 때 어떻게 처리하는가? -> 한 번에 전체 등록 (공통 메타데이터 폼 제공)
- [완료] GeoJSON properties 처리 방법은? -> 자동 매핑 (우선순위 기반). 매핑 실패 시 빈칸
- [완료] 임포트 시 기존 장애물과 중복 검사가 필요한가? -> 중복 검사 없이 그대로 등록
- [완료] MultiPoint, MultiLineString, MultiPolygon 등 Multi 계열 geometry도 지원해야 하는가? -> 미지원. Point, LineString, Polygon 3종만 지원. Multi 계열은 에러 메시지 표시
- [완료] 임포트 전에 개별 Feature의 속성을 수정할 수 있어야 하는가? -> 개별 수정 불가. 공통 메타데이터 폼으로 일괄 적용만 지원
